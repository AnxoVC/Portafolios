##  Características

* **Totalmente Responsivo:** Adaptado para una navegación perfecta en móviles, tablets y ordenadores.
*  **Carga Ultra Rápida:** Optimizado gracias al uso de Vite y componentes eficientes.
*  **Interfaz Moderna:** Estética limpia utilizando Tailwind CSS para un acabado profesional.
*  **Showcase de Proyectos:** Galería interactiva con detalles de mis desarrollos más recientes.
*  **Sección de Contacto:** Enlaces directos para facilitar la comunicación y networking.
##  Tecnologías
* **HTML5:** Estructura de la terminal.
* **CSS3:** Estilizado (fuentes monoespaciadas y fondo oscuro).
* **JavaScript:** Lógica para simular la escritura y comandos.
